# The Highway Pursuit

## Game Description

**The Highway Pursuit** is a simple graphics project created using the **iGraphics** library in C. The game is a single-player 2D car chase game where the player controls a car driving along a straight highway. A police car continuously chases the player, and the game ends if the police car collides with the player. Enemy cars and obstacles appear on the road, increasing the challenge. The player can shoot enemy vehicles using limited ammunition and collect special power-ups such as speed boosts and ammo upgrades to survive longer.

## Features
•	Player-controlled car movement
•	Continuous police chase
•	Enemy cars and obstacles
•	Shooting system with limited ammo
•	Power-ups such as speed boost and ammo upgrade
•	Scoring based on survival time and destroyed enemies

## How to Run the Project

Make sure you have the following installed:
- **Visual Studio 2013**
- **MinGW Compiler** (if needed)
- **iGraphics Library** (included in this repository)


Open the project in Visual Studio 2013
- Open Visual Studio 2013.
- Go to File → Open → Project/Solution.
- Locate and select the .sln file from the cloned repository.
- Click Build → Build Solution
- Run the program by clicking Debug → Start Without Debugging


## How to Play

### **Controls**
| User       | Move Left | Move Right | Go Forward       | Go reverse | Fire |  |
|-------------|----------|-----------|-----------|-------|------|-------|
| **Car** | `A / Left Arrow`      | `D / Right Arrow`       | `W / Up Arrow`       | `S / Down Arrow`   | `Space Button`  |    |



### **Game Rules**

- Different health points for each level
- Score increases by fulfilling different game criteria.
- Each time a bot car is destroyed by the user a specific point is added to the score.
